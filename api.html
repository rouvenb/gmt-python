    Low-level wrappers for the GMT C API¶

    The GMT C API is accessed using ctypes. The gmt.clib package offers functions and classes that wrap the C API with a pythonic interface.

    clib.call_module(session, module, args) Call a GMT module with the given arguments.
    clib.create_session([name]) Create the GMTAPI_CTRL struct required by the GMT C API functions.
    clib.destroy_session(session) Terminate and free the memory of a registered GMTAPI_CTRL session.
    clib.load_libgmt([libname]) Find and load libgmt as a ctypes.CDLL.
    clib.get_constant(name) Get the value of a constant (C enum) from gmt_resources.h
    clib.APISession() Context manager to create a GMT C API session and destroy it.
