    gmt.clib.load_libgmt(libname='libgmt')[source]¶

    Find and load libgmt as a ctypes.CDLL.

    If not given the full path to the library, it must be in standard places or by discoverable by setting the environment variable LD_LIBRARY_PATH.

    Parameters:

    libname : str

    The name of the GMT shared library without the extension. Can be a full path to the library or just the library name.
    Returns:

    ctypes.CDLL object

    The loaded shared library.
