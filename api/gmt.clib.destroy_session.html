    gmt.clib.destroy_session(session)[source]¶

    Terminate and free the memory of a registered GMTAPI_CTRL session.

    Warning

    Best not used directly. Use gmt.clib.APISession instead.

    The session is created and consumed by the C API modules and needs to be freed before creating a new. Otherwise, some of the configuration files might be left behind and can influence subsequent API calls.

    Parameters:

    session : C void pointer (returned by ctypes as an integer)

    The active session object produced by gmt.clib.create_session.
